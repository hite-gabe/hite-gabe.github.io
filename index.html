An ultra-lightweight, legacy screen recorder engineered specifically for older hardware profiles and X11 display servers.

Download .deb Package

Built for Older Devices

Designed strictly for legacy environments. OpenRec has minimal memory overhead and resource consumption, making it ideal for older machines and lower-spec hardware configurations.

Zero Bloat

Written cleanly using Python 3 and native PyQt6. Free from heavy background tracking services, modern application wrapper overhead, or web framework resource drains.

Audio Capture

Includes basic, light UI toggles for monitoring traditional Linux systems or microphone hardware feeds through FFmpeg pipelines.

Quick Installation

Download the architecture configuration openrec.deb file, open your shell inside the download path, and pass it to your local packages manager toolchain:

sudo apt update
sudo apt install ./openrec.deb

Supported Operating Systems

Distribution / OS Display Server Requirement Compatibility Status
Linux Mint (Legacy or Modern) X11 Session (Cinnamon, XFCE, MATE) Fully Supported ✅
Ubuntu / Debian variants X11 Desktop / Legacy environments Fully Supported ✅
Modern Wayland sessions Any environment using Wayland display Not Supported (Requires X11 fallback) ❌
